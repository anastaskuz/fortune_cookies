Для начала работы:

1. Создайте виртуальное окружение
python -m venv <venv_bot>
2. Активируйте виртуальное окружение
<venv_bot>\Scripts\activate
3. Установите зависимости из файла requirements.txt
pip install -r requirements.txt
4. Скачайте папку img в папку проекта по ссылке
https://drive.google.com/drive/folders/1P2CU99yHVSMJaFOpU7jkpoQY02K-RztK?usp=drive_link
5. Внесите данные в файл run_github.bat
Токен бота
Имя базы данных
Путь к папке, в которой будет создана база данных
путь к папке с картинками

Для запуска бота:
Запустить файл run_github.bat и не закрывать cmd
